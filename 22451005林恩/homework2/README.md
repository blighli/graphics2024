# 作业二：太阳系

## 项目概览

本项目使用 OpenGL 创建一个简单的太阳系模型，包括太阳系中的八大行星和太阳九个天体，具体有以下功能：

- **纹理映射**：给各个天体进行了纹理映射。
- **光照效果**：太阳为光照效果
- **交互控制**：可以控制视角。
- **显示天体名字**：可以显示天体的名字。
